Don't forget to set the URSEL bit in the UCSRC register to '1'.
The code should read as -
UCSRC = (1<<URSEL) | (1<<USBS) | (3<<UCSZ0);

This should be done for both the transmitter and the receiver.

See p165 of the ATmega32A datasheet for further details.﻿
